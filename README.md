## 📖 About

67 Sec Optimization Pack is a collection of carefully selected Windows optimizations designed to improve overall system responsiveness and gaming performance.

The pack automatically detects your GPU vendor and applies the appropriate GPU optimizations while also applying network, latency, cleanup, and Windows performance tweaks.

Everything is bundled into a simple one-click package that can be applied in under a minute.

## ⚡ Usage

### Method 1 (Recommended)

1. Download the latest release
2. Extract the ZIP
3. Run `Apply The Pack.cmd` as Administrator
4. Wait for the optimization process to complete
5. Restart your PC

### Method 2

Clone the repository:

```bash
git clone https://github.com/ToXTweaks/Optimization-Pack.git
```

Then run:

```text
Apply The Pack.cmd
```

as Administrator.
